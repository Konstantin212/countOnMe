# CountOnMe Documentation

## Structure

- **features/** — User-facing feature documentation
- **api/** — Backend API endpoint documentation
- **architecture/** — Architecture decisions and data flow

## Generating Docs

Use the `/doc` command to generate or update documentation for a feature area:

```
/doc product management
/doc meal builder
/doc device auth
```

Documentation is generated by the `doc-writer` agent from source code.
